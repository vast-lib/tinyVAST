# Data to demonstrate model-based diet proportions

Data to estimate predator-expanded stomach contents (PESC), i.e., a
multi-prey single-predator model for stomach contents of red grouper in
the Gulf of Mexico that also estimates biomass density for red grouper.
Diet proportions are then the product of predicted diet proportions and
prey specific consumption, normalized across prey categories. Copied
from VAST data set `PESC_example_red_grouper`

## Usage

``` r
data(red_grouper_diet)
```
