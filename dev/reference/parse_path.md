# Parse path

`parse_path` is copied from `sem::parse.path`

## Usage

``` r
parse_path(path)
```

## Arguments

- path:

  character string indicating a one-headed or two-headed path in a
  structural equation model

## Value

Tagged-list defining variables and direction for a specified path
coefficient
