# Condition and density example

Data used to demonstrate and test a bivariate model for morphometric
condition (i.e., residuals in a weight-at-length relationship) and
density for fishes, using the same example as was provided as a wiki
example for VAST. Data are from
[doi:10.3354/meps13213](https://doi.org/10.3354/meps13213)

## Usage

``` r
data(condition_and_density)
```
