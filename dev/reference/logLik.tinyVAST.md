# Extract the (marginal) log-likelihood of a tinyVAST model

Extract the (marginal) log-likelihood of a tinyVAST model

## Usage

``` r
# S3 method for class 'tinyVAST'
logLik(object, ...)
```

## Arguments

- object:

  output from `tinyVAST`

- ...:

  not used

## Value

object of class `logLik` with attributes

- val:

  log-likelihood

- df:

  number of parameters
